About the Closed Sales Prototype

This prototype is a remake of the FDIC Closed Sales application, the current version of which is located at: https://sales.fdic.gov/closedsales/LoanSales.asp. Our goals with the prototype are to demonstrate a better user experience, and to open up the underlying data.

User Experience

The user experience is streamlined. Instead of two screens - a search form and results page - we combined the search/filter and results into a single page. Users get instant access to information, and then can refine filters and see results instantly.

Open Data

The current system makes data available via a web interface. This is a great start, but it is difficult to export the data for analysis (e.g. in Excel) or use in another application. For the Closed Sales prototype, we illustrate the following features:

  • Bulk downloads – Users (or other systems) can download bulk datasets in CSV or JSON formats for analysis or building applications using the data.
  • API-first design – The user interface is built on top of an API. The same API can be used by third-party applications to make use of the data, including the same sorts of filter and sort features incorporated into the UI.
  • Developer documentation – We provide interactive developer documentation, making it easy for developers to understand the API, and to build/test queries against it.
